# Directive: Get ClickUp Task

This directive instructs the agent to retrieve a task from ClickUp.

## Command

Use the `clickup_get_task.py` script to get a task.

### Arguments

- `task_id`: The ID of the task to retrieve.

### Example

```bash
.venv/bin/python3 execution/clickup_get_task.py "987654321"
```
