# Directive: Delete ClickUp Task

This directive instructs the agent to delete a task from ClickUp.

## Command

Use the `clickup_delete_task.py` script to delete a task.

### Arguments

- `task_id`: The ID of the task to delete.

### Example

```bash
.venv/bin/python3 execution/clickup_delete_task.py "987654321"
```
